	\subparagraph{Certificados - Suspensión}

	En este apartado, es necesario tener configurada la CRL, para ello es necesario que mediante el rol administrador de seguridad se configure.

	En esta opción, el sistema permite suspender certificados, para ello presionamos el botón Buscar de la pantalla inicial de suspensión de certificado (Fig. \ref{caroot_fig91}) y nos aparecerá el listado de certificados (Fig. \ref{caroot_fig92})  (todos, si no asignamos ningún criterio, en caso contrario los filtrará según las propiedades que completemos en el formulario), si en la búsqueda presionamos (\includegraphics[scale=1]{./img/icono_ver_cuerpo.png}), podemos ver el cuerpo del certificado (Fig. \ref{caroot_fig93}), si presionamos sobre \textit{Suspender}, el sistema nos muestra una pantalla para suspender el certificado (Fig. \ref{caroot_fig94}) en esta ventana también podemos descargarnos el certificado, presionamos \textit{Continuar} y el sistema nos muestra una nueva pantalla con el certificado suspendido (Fig. \ref{caroot_fig95}), presionamos \textit{Continuar} y volvemos a la página de inicio.

	\ifthenelse{\equal{\jobname}{\detokenize{ra}}}{
		Una vez realizada la operación, el sistema envía un email con el cuerpo del email que hayamos configurado mediante el rol Administrador de Seguridad, en el apartado Administración ->\ Configuración de las notificaciones.
	}{}
