	\subparagraph{Certificados - Revocación}

	Para poder llevar a cabo esta operación, es necesario tener configurada la CRL, para ello es necesario que mediante el rol administrador de seguridad se haya configurado.

	En este apartado, el sistema permite revocar certificados, para ello presionamos el botón \textit{Buscar} de la pantalla inicial de revocación de certificado (Fig. \ref{caroot_fig86}) y nos aparecerá el listado  de certificados (Fig. \ref{caroot_fig87})  (todos si no asignamos ningún criterio, en caso contrario, filtrará según las propiedades que completemos en el formulario), si en la búsqueda presionamos (\includegraphics[scale=1]{./img/icono_ver_cuerpo.png}), podemos ver el cuerpo del certificado (Fig. \ref{caroot_fig88}), si presionamos sobre \textit{Revocar}, el sistema nos muestra una pantalla (Fig. \ref{caroot_fig89}), en la cual hay que seleccionar el motivo de revocación, en esta ventana también podemos descargarnos el certificado, una vez seleccionado el motivo presionamos \textit{Continuar} y el sistema nos muestra una nueva pantalla con el certificado revocado (Fig. \ref{caroot_fig90}) , presionamos \textit{Continuar} y volvemos a la página de inicio.

	\ifthenelse{\equal{\jobname}{\detokenize{ra}}}{
		Una vez realizada la operación, el sistema envía un email con el cuerpo del email que hayamos configurado mediante el rol Administrador de Seguridad, en el apartado Administración ->\ Configuración de las notificaciones.
	}{}
